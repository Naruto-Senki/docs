> **此栏目为目前已经确认的问题或bug，同时将罗列个版本反馈的问题。**
>
> 1. 如有新问题，请点击`群文件`里`在线文档`内的`反馈文档`进行反馈。
>
> 2. 如对哪条问题有疑义或补充，请在群里`@Re`或在`反馈文档`中对应问题的编号后写下。
>
> 3. 如发现哪条问题不存时，则说明该问题以重复或以被合并到其他问题内，请注意翻阅查看。
>
> **反馈的具体格式请参考下面的`反馈问题格式`。**
>
> 此栏目将不定时更新，建议早上阅读，因为通常会在晚上更新。
> 此栏目内容也会与V2群内的`反馈文档`同步更新。

???+ note "反馈问题格式"

    反馈问题，如涉及到战斗逻辑，应描述如下信息：

    1. 游戏模式（如果在进入彩蛋后出现问题，应特殊标注）

    2. 对局角色：主要为你操作的角色（若可能受其他角色影响，应记录相应角色名）

    涉及到技能，请按照普攻（普通攻击）、技能1-5、奥义1-2格式进行填写（阿拉伯数字或中文数字皆可）。

    如UI、贴图或角色产生偏移、错位等问题，最好提供错误时的截图以及简要描述。

    如问题难以描述，最好在反馈文档中贴上截图，或发送一段视频到V2群里，并`@Re`。

    做好反馈描述，不仅有利于开发者进行定位修复，
    也利于其他遇到相似问题的玩家参考和进行补充。

??? note "标签说明"

    <font color=#00c853>:material-check:</font>：表示此问题或bug已经解决。

    <font color=#FF5252>【待修复】</font>：等待下一版本进行修复。

    <font color=#7B1FA2>【待排查】</font>：此问题等待开发者排查中。

    【xx待完善】：xx需要后续完善。

    【反馈描述待完善】：需要提问者增加更详细描述信息或图片。

    【未知】：不明原因。

## v2.1.0 已知问题

!!! question "1. 选人界面人物语音播放不完全，像是被掐断一样"

    【音频待完善】

    旧版音频素材问题，以前为了优化音频占用内存，部分音频有分割掉最后一部分。

!!! question "2. 安装包名不是旧版，而是共存，这样会让很多玩家丢失存档，并且失去情怀"
