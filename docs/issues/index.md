???+ note "反馈问题格式"

    反馈问题，如涉及到战斗逻辑，应涵盖以下部分：

    1. 游戏模式（如果在进入彩蛋后出现问题，应特殊标注）

    2. 对局角色：主要为你操作的角色（若可能受其他角色影响，应记录相应角色名）

    涉及到技能，请按照普攻（普通攻击）、技能1-5、奥义1-2格式进行填写（阿拉伯数字或中文数字皆可）。

    涉及到游戏内文字、技能描述等错误或不准确，请在`建议文档`中增加新条目，并可以附上修补方案。

    若UI、贴图或角色产生偏移、错位等问题，最好提供错误时的截图以及简要描述。

    如需引用已存在的问题，请使用 **# + 问题编号** 格式（例如# 15）。

    如问题难以描述，最好在反馈文档中贴上截图，或发送一段视频到V2群里，并要写清`问题编号`，并`@Re`。

    做好反馈描述，不仅有利于开发者进行定位修复，
    也利于其他遇到相似问题的玩家参考和进行补充。

??? note "标签说明"

    <font color=#00c853>:material-check:</font>：表示此问题或bug已经解决。

    <font color=#FF5252>【待修复】</font>：等待下一版本进行修复。

    【待排查】：此问题等待开发者排查中。

    【反馈描述待完善】：需要提问者或其他遇到此问题的玩家增加更详细描述信息或图片。

    【未知】：不明原因 或 无解。
