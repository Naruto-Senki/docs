## 模式

!!! tip "提示"

    当该模式可自选角色时，背景将是红色，如1v1。

    在所有模式中，默认不选择角色时均视为随机选择。

- 1v1：1v1对决，角色满状态，满金币（主要用于测试）
- 3v3：3v3常规对局`（启用忍具）`
- 4v4：4v4常规对局`（启用忍具）`
- Hardcore：4v4硬核模式`（禁用忍具）`
- Clone：克隆模式，3v3，一方的角色完全相同

!!! note "注释"

    当前默认所有模式均开启守护忍。

    游戏进入彩蛋后，部分模式可能结算界面不会显示完全，且无法获得金币。
