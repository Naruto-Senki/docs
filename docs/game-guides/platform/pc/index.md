## 按键说明

- 全局返回/暂停游戏：ESC
- 切换窗口：F11

### 基本操作

- 移动：WSAD
- 攻击：J
- 技能 1：U
- 技能 2：I
- 技能 3：O
- 奥义 1：K
- 奥义 2：H
- 拉面：L
- 刃具 1：B
- 刃具 2：N
- 刃具 3：M

### 刃具商店

- 打开刃具商店：SPACE（空格键）
- 选择刃具：1-9
- 购买刃具：B

## 特殊功能

使用F11按键可以切换窗口大小，具体如下：

1. 1280x720

2. 1600x900（默认窗口大小）

3. 全屏

!!! warning "警告"

    在某些Linux发行版上，切换窗口可能导致UI位置错误，可以修改`script/window.lua`文件内的窗口参数达到相同效果。
