## 按键说明

- 全局返回/暂停游戏：++esc++
- 切换窗口：++f11++

### 基本操作

- 移动：++w++ ++s++ ++a++ ++d++
- 攻击：++j++
- 技能 1：++u++
- 技能 2：++i++
- 技能 3：++o++
- 奥义 1：++k++
- 奥义 2：++h++
- 拉面：++l++
- 忍具 1：++b++
- 忍具 2：++n++
- 忍具 3：++m++

### 忍具商店

- 打开忍具商店：++space++（空格键）
- 选择忍具：++1++ - ++9++
- 购买忍具：++b++

## 特殊功能 { id="special_feature" }

使用++f11++按键可以切换窗口大小，具体如下：

1. 1280x720

2. 1600x900（默认窗口大小）

3. 全屏

!!! warning "警告"

    在某些Linux发行版上，切换窗口可能导致UI位置错误，可以修改`script/window.lua`文件内的窗口参数达到相同效果。
