## How to play

- Back & Pause game: ++esc++
- Switch window size: ++f11++

### Basic

- Movement: ++w++ ++s++ ++a++ ++d++
- Attack: ++j++
- Skill 1: ++u++
- Skill 2: ++i++
- Skill 3: ++o++
- Ougi 1: ++k++
- Ougi 2: ++h++
- Ramen: ++l++
- Gear 1: ++b++
- Gear 2: ++n++
- Gear 3: ++m++

### Gear Shop

- Open shop: ++space++
- Select gear: ++1++ - ++9++
- Buy gear: ++b++

## Special Feature

use ++f11++ to swtich the window size:

1. 1280x720

2. 1600x900 (default)

3. Fullscreen

!!! warning

    In some Linux, switching window sizes can cause UI position issues,

    you can modify the `script/window.lua` file to achieve the same effect.
