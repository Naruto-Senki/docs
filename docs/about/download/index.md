!!! question "为什么一个app会分出多个架构的版本？"

    在移动端，只选择对应架构的app，可以减少10-30+MB的大小，且同时[蓝奏云](https://lanzou.com)有100MB上传限制。

    （缺点是app在传输至不支持此架构的设备上将无法运行，此时应选择通用版本，即包含了所有架构的app）
