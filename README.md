> Naruto Senki: V2的文档及更多内容.

此文档使用Markdown编写，由[mkdocs](http://www.mkdocs.org)编译运行.

## 运行

* Python-3.7+

```
pip install -r requirements.txt
python -m mkdocs serve
```

或

### pipenv

```
pip install pipenv
pipenv install
pipenv run python -m mkdocs serve
```
